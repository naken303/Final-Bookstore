วิธีการติดตั้ง download zip file ทั้ง project ไปแล้ว แตกไฟล์ ตั้งชื่อ folder ว่า "bookstore" และทำตามลำดับต่อไปนี้
1. เปิด Visual Studio Code และ open folder "bookstore"
2. เปิด Terminal ใน vscode และพิมพ์ว่า npm i จากนั้นรอให้ติดตั้งโดยสมบูรณ์
3. เปิด mongodb และสร้าง database ชื่อว่า bookstore
4. สร้าง collection ใน bookstore ดังนี้ 1.books 2.carts 3. orders 4.users
5. ทำการ import ไฟล์ที่อยู่ใน project folder ที่ชื่อ data_for_import_mongo โดย import ให้ชื่อไฟล์ตรงกับชื่อ collection ที่จะ import
6. สิทธิ์จะถูกแบ่งออกเป็น 2 สิทธิ์ user และ admin
  6.1 user จะสามารถ sign up และทำการ login เองได้
  6.2 admin เริ่มต้นที่มีให้ email = "a@a", password = "1"

7. พิมพ์ "npm run dev" ใน Terminal ของ vscode เพื่อ run website
8. เปิด browser และพิมพ์ url ว่า http://localhost:3000/
9. หน้าเว็บจะถูกเปิดโดยสมบูรณ์

คำสั่งของพี: ให้พีทำขั้นตอนที่กล่าวมาข้างต้น และ เข้าสู่หน้า admin (ขั้นตอน 6.2 คือ email และ password ที่ต้องใส่) เพื่อทำรายการเพิ่มหนังสือ สามารถ click ที่ปุ่ม add new book ใน menu book lists ได้
          โดยจะต้องเพิ่มหนังสือ 40 เล่ม โดยรายละเอียดและภาพปกหนังสือจะต้องครบถ้วนสมบูรณ์ จำนวนหนังสือสามารถ สุ่มขึ้นมาเองได้ แต่ประเภทหนังสือ
          จะต้องไม่น้อยกว่า 8 ประเภท และแต่ละประเภทจะต้องมีหนังสือไม่น้อยกว่า 5 เล่ม โดยมีหมวดหมู่เริ่มต้นมาให้แล้ว 3 ประเภทได้แก่
          1. "นิยายสืบสวน สอบสวน" 2. "จิตวิทยา" 3. "ฮีลใจ"
          * โดยสรุปคือ พีจะต้องเพิ่มหนังสือ 8 ประเภท แต่ละประเภทประเภทไม่น้อยกว่า 5 เล่ม
          * สิ่งที่ต้องระวังคือ text ประเภทหนังสือในแต่ละเล่ม ของแต่ละประเภท จะต้องเหมือนกัน 100% ไม่อย่างนั้นระบบจะถูกจัดให้เป็นหมวดหมู่ใหม่ ตัวอย่าง
            หมวดหมู่ ฮีลใจ กับ ฮีลลลใจ จะกลายเป็น 2 หมวดหมู่แทน
          โดยหลังจากทำตามคำสั่งข้างต้นเสร็จสิ้นให้ export ข้อมูลในรูปแบบ json ของ collection books และ folder "books_cover" ที่จะอยู่ใน public/images ขึ้น google drive และแชร์เป็นสาธารณะให้
          กับคนให้กลุ่ม download เพิ่อจะได้ import เป็นข้อมูลของตัวเอง และนำไปอัดวิดีโอนำเสนอได้อย่างราบรื่น
          
          หลังจากพีทำทุกขั้นตอนเสร็จสิ้นแล้วจะมีการ update ข้อมูลภายใน github ในส่วนของ folder "data_for_import_mongo" และไฟล์ข้อมูลที่จะให้ไป import ในแต่ละ collection ดังข้อ 5. ข้อมูลของทุกคนจะได้เหมือนกัน
          โดยก่อนจะ import ให้ delete ข้อมูลทุก record ที่ทุกคนเล่นไว้ก่อนหน้าออกให้หมดใน mongodb (ทำไปเพื่อความราบรื่นในการทำตามบทที่เขียนให้)
          และจะมีไฟล์เพิ่มเข้ามาอีกไฟล์ ใน githup คือบทพูดของแต่ละคนหลังจากที่พีส่ง json collection และ folder "books_cover" แล้วเท่านั้น

คำสั่งของคนอื่น: ตอนนี้ลองเล่นติดตั้งและเล่น website ดูก่อนตามข้อ 1 ถึง 9 และรอบทพูดของตัวเองใน github หากอัพแล้วจะมีการแจ้งให้ทราบในกลุ่ม
